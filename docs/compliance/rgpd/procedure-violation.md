# Procédure de Notification de Violation de Données

## 1. DÉTECTION

Tout membre de l'équipe suspectant une violation doit immédiatement :

1. Contacter le DPO : dpo@zaksoft-creations.com
2. Documenter l'incident (heure, périmètre, données concernées)

## 2. ÉVALUATION

Le DPO évalue dans les 24h :

- Nature des données compromises
- Nombre de personnes concernées
- Risques pour les droits et libertés

## 3. NOTIFICATION CNIL

**Délai :** 72h après prise de connaissance

**Contenu :**
- Description de la violation
- Catégories et nombre de personnes concernées
- Mesures prises / envisagées

## 4. COMMUNICATION AUX PERSONNES CONCERNÉES

**Cas obligatoire :** Risque élevé pour les droits et libertés

**Contenu :**
- Description de la violation
- Recommandations (changer mot de passe, etc.)
- Coordonnées du DPO
